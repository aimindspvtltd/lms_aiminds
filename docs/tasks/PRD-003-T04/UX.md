## Mobile Adaptations

### Content Type Selection (Mobile)
- Stack cards vertically
- Full width cards
- Larger touch targets

### Video Player (Mobile)
- Full width video
- Native controls
- Landscape support
- Picture-in-picture

### Document Viewer (Mobile)
- Single column layout
- Pinch to zoom
- Horizontal scroll for code blocks
- Collapsible sections

## Accessibility

### Keyboard Navigation
- Tab through all interactive elements
- Enter/Space to activate
- Arrow keys for reordering
- Escape to cancel modals

### Screen Reader Support
- Descriptive labels for icons
- Announce content type
- Progress state announcements
- Form validation errors

### Visual Accessibility
- High contrast mode support
- Focus indicators
- Sufficient color contrast
- No color-only indicators

### Lazy Loading
- Load video thumbnails first

- Local storage for drafts
